Connect your own Oura, Apple Health & calendars

This dashboard is a static site with no server, so every connection below is either a live public feed or a small file you keep updated yourself — nothing that requires storing a private API key in this public repo.

Calendars — live

  1. In Google Calendar, open Settings for the calendar you want to add.
  2. Under Access permissions, turn on Make available to public (only for calendars you're fine with anyone-with-the-link seeing).
  3. Scroll to Integrate calendar and copy the src= URL from the embed code.
  4. Add it to _data/calendars.yml as a new name / embed_url pair.

Weather & UV — live

Fetched automatically from the free, keyless Open-Meteo API. Just set your coordinates once in _data/location.yml.

Oura Ring — manual

Oura doesn't offer a public/keyless API, so open the Oura app each morning and copy today's scores into _data/oura.yml.

Apple Health — manual

Apple Health data lives on your device with no public web API, so open the Health app and copy today's numbers into _data/apple_health.yml. Give a metric a goal to show it as a progress bar.

Built with GitHub Pages · only calendars you've explicitly made public are shown here.
